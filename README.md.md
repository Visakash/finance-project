# Finance Microservices System

## Overview
This project is a microservices-based finance system built using Spring Boot and Spring Cloud.

## Architecture
- Eureka Server (Service Discovery)
- API Gateway
- Loan Service
- Credit Check Service
- Customer Service
- Document Service
- Notification Service

## Technologies
- Java
- Spring Boot
- Spring Cloud
- Kafka
- MySQL
- AWS S3
- Maven
